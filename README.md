# ☕ Coffee Machine Simulator (Python)

A simple terminal-based Coffee Machine Simulator built using Python and Object-Oriented Programming (OOP).

This project simulates a real coffee vending machine where users can order drinks, make payments, manage machine resources, and receive change. It was built as a learning project to practice Python fundamentals, classes, functions, and program flow.

---

## ✨ Features

* ☕ Order from three drinks:

  * Latte
  * Espresso
  * Cappuccino
* 💰 Payment system with:

  * Multiple payments supported
  * Change calculation
  * Payment cancellation
* 📦 Resource management

  * Water
  * Milk
  * Coffee
  * Sugar
* 🔍 Resource availability check before preparing coffee
* 🔄 Automatic resource deduction after every successful order
* 🛠️ Machine refill option when resources are insufficient
* 📊 Machine report showing:

  * Remaining resources
  * Current balance
* ❌ Input validation for invalid menu and payment entries
* 🎬 Terminal animations and loading effects for a better user experience
